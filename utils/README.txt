PASTA UTILITÁRIOS
-----------------

O objetivo dessa pasta é guardar os instaladores das ferramentas utilitárias
que usamos para desenvolver o projeto e documentar a forma de instalá-los.

UTILITÁRIOS:
------------

Astah Community 6.4:
	Programa de modelagem UML desenvolvido em JAVA. É o antigo JUDE.
	
	- Instalação: dpkg -i astah-community_6.4_all.deb
        - execução: Acionar o íconde do programa no menu Aplicativos > Acessórios

SQL Power Architect:
	Programa de modelagem E-R que permite integração com o PostgreSQL.

	- Instalação: descompacte o arquivo SQL-Power-Architect-generic-jdbc-1.0.6.tar.gz em uma pasta acessível a todos os usuários (ex: /opt) com o comando tar zxvf SQL-Power-Architect-generic-jdbc-1.0.6.tar.gz
	-Execução: java -jar architect.jar
